# Samsung Supply Chain Dashboard

This project presents an end-to-end Supply Chain & Logistics dashboard built using Power BI, focusing on key business stages — Supplier, Inventory, Shipment, and Customer.

## Overview
- The dashboard provides a consolidated view of operational performance by tracking key metrics such as revenue, profit, inventory value, shipment quantity, and delivery performance. It is designed to highlight patterns, identify potential delays, and support better decision-making across the supply chain.

## Key Learnings
- Implemented parameterization to efficiently manage and update data sourcing with minimal changes
- Developed calculated measures to understand relationships and interactions across different datasets
- Structured data using fact and dimension tables to enable scalable and flexible analysis
- Designed a cohesive dashboard layout to represent the flow of supply chain operations

## Dataset
The project uses a combination of fact and dimension tables including:
- Supplier, Customer, Product, and Facility data
- Inventory, Procurement, and Production datasets

## Tools Used
- Power BI
- Data Modeling
- DAX (Data Analysis Expressions)

## Note
- This project was created to explore data modeling, transformation, and visualization techniques in a supply chain context.
